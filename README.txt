Displays recent posts thumbnails as the masonry grid gallery.

== Description ==

Gallery Recent Posts is a widget that can come in handy for many site. Display your recent posts in more creative and inovative way - with the popular masonry grid gallery. 

All you need to do is to install the plugin and select number of posts to show and number of columns. Gallery Recent Posts widget will automatically get the first n-number posts with thumbnails and sort them in the masonry grid gallery. 

You can use the gallery on your sidebar, footer or anywhere the theme you are using have widget area.

== Installation ==

1. Use native WordPress Plugin Installer or Download and unzip plugin to "/wp-content/plugins/" directory
2. Activate plugin in "Plugins" Menu
